    　关于硕正报表
    
　　如何能不用编程就让实施人员、最终用户
随心所欲地编排报表、引用数据、打印报表、
设计票据套打，一直是程序员们的梦想。

　　Supcan Report (硕正报表) 使这一梦想
成真。和Excel 等电子表不同的是，硕正报表
侧重于面向企业应用开发，侧重于和企业的
Web应用产品的集成。

　　Treelist、Freeform主要用途是实现对数
据的增、删、改操作，而 Report 则仅仅是展
现、输出数据。

　　Supcan Report具备报表的一般性功能：
任意输入、复制粘帖、Undo、字体、对齐、
颜色、更改单元格样式、组合单元格、浮动的
文字、图片、斜线、统计图、内置公式函数、
表达式解析、自动计算、文字超宽处理、分类
汇总、交叉表、筛选过滤、定制分页、多工作
表等等。

　　Supcan Report有超强的打印输出功能，
例如分页、套打、缩放、精确打印等，也可以
所见所得地将报表转换成PDF、JPG、PNG、
Excel、HTML等格式的文件。

　　从中心数据源取数填充，是硕正报表最重
要的功能。按照硕正公司数据与展现分离的
理念，程序员的任务只是将可能会被 "使用" 
的数据编入中心数据源目录、实现被取数的功
能即可，至于如何被使用、如何展现，已不必
非要让程序员来做。

　　Supcan Report 也允许从其它的 XML、
JSON、Web Service、HTML Table等数据
源取数。

　　和 Supcan Treelist、Freeform 一样，
Supcan Report 的存储格式也是XML，程序
员用Notepad也能作一些简单的改动。

　　此外，硕正报表提供了自定义工具条、自
定义菜单以及丰富的API。

    40.指标库 借助预设的“指标库”，采用鼠标直接拖拽，大大提高了上传报表的设计效率.
    41.内嵌表格 演示了内嵌表格的“指标库”，用鼠标可一次性拖入一个区块.
    42.内嵌树列表 演示了内嵌Treelist的“指标库”，用以弥补Report交互输入不及Treelist的缺陷.
