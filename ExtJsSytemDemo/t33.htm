1.所谓“定制”是指用户作了更改列宽、调整列位置、排序、选择显示列、分屏冻结、打印预览等操作后，希望能将这些外观和打印配置的信息保存起来，以便下次打开本页面时自动套用；
2.任意更改列宽、调整列位置、或排序、或切换到树，然后点击"保存定制(模拟)"；
3.再任意作上述操作，然后点击"应用定制(模拟)"；

  


1.所谓“模拟”是指并非真正保存定制信息，实际使用中应该将定制信息通过Http POST给应用服务器，即保存到服务器；
2.源码分析：
<script language="JavaScript">
 //用页面变量存放定制信息，来模拟服务器存放
 var g_custom;

 //应用定制
 function setcustom()
 {
  AF.func("setCustom", g_custom);
 }

 //取得定制信息
 function getcustom()
 {
  g_custom = AF.func("getCustom","");
  //AJAX 上传到服务器
  alert(g_custom);
 }
