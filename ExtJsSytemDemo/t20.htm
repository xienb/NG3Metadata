2.源码分析：“模拟提交”按钮执行的js如下：
function postModify()
{
 var xml = AF.func("GetChangedXML", "");
 if(xml==null || xml=="")
  AF.func("MessageBoxFloat", "请先作增、删、改操作!	");
 else if(xml!="0")
  alert(xml);
}
从源码可以看出，该js只是调用了GetChangedXML()获得一个XML串而已，该XML大串包含了当前的全部数据。要实现真正的提交，一般采用 AJAX 技术把该XML串 POST 给后端的应用服务器。
