该方案没有采用外部的分页器，而是采用了Treelist内部的垂直滚动条 即拖即刷新 的机制.
1.共500行，每次取100行，请用鼠标抓住右侧滚动条不要松开，慢慢向下拖动，遇到空白再松开，查看其刷新效果:
2.如果点击列头排序、以树展现，将自动获取全部数据;



1.本案例没有绑定外部分页器，且使用了上一个demo页相同的的XML描述文件。页面js如下：
function OnReady(id) {
 AF.func("Build", "treelist/t1.xml");
 AF.func("Load", "http://www.supcan.cn/DotNet/access.aspx?startRow=@startRow&rows=@rows \r\n mode=Asynch");
}
2.如果你希望将这二个属性在XML中写死，那么将XML描述文件改成如下即可：
<Properties dataURL="http://www.supcan.cn/DotNet/access.aspx?startRow=@startRow&rows=@rows" />

使用够简单吧！
